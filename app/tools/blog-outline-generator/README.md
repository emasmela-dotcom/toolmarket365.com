# Blog Outline Generator

## What It Does
Generates detailed blog outlines with H2, H3 headings and bullet points. Works with OpenAI API for AI-powered outlines, or generates template outlines without an API key.

## How to Use

1. **Enter blog title** (required)
2. **Select tone:**
   - Professional
   - Casual
   - Friendly
   - Academic
   - Conversational
3. **Choose level:**
   - Beginner
   - Intermediate
   - Advanced
4. **Add primary keyword** (optional) - for SEO focus
5. **Add OpenAI API key** (optional) - for AI-powered outlines
6. **Click "Generate Outline"**

## Expected Outcome

### With API Key:
- **AI-generated outline** tailored to your title, tone, and level
- **Structured format** with H2 and H3 headings
- **Bullet points** for each section
- **SEO-optimized** if keyword provided
- **Tone-appropriate** content structure

### Without API Key:
- **Template outline** with standard blog structure
- **7 main sections:**
  - Introduction
  - Understanding the Basics
  - Key Concepts and Strategies
  - Advanced Techniques
  - Real-World Applications
  - Common Pitfalls to Avoid
  - Best Practices
  - Conclusion
- **Ready to customize** with your content

## Output Format

```
# Blog Title

## Introduction
- Hook: ...
- Context: ...
- Thesis: ...

## Section Name
- Point 1
- Point 2
- Point 3

...
```

## Features

- **Works offline** - Template generation doesn't require API
- **AI-powered** - Optional OpenAI integration for better results
- **Copy to clipboard** - One-click copying
- **Multiple tones** - Adapts to your writing style
- **Level-appropriate** - Beginner to advanced content

## Tips

- Use specific, descriptive titles for better outlines
- Add keywords for SEO-focused structure
- Template outlines are great starting points
- AI outlines are more detailed and tailored
- Copy outline to your writing tool for easy reference

## Getting an OpenAI API Key

1. Go to https://platform.openai.com/
2. Sign up or log in
3. Navigate to API Keys section
4. Create a new secret key
5. Copy and paste into the tool
