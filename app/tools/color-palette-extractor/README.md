# Color Palette Extractor

## Purpose
Extract the dominant color palette from any image. Perfect for identifying brand colors, creating matching color schemes, or analyzing the color composition of images for design purposes.

## How to Use

1. **Upload an Image:**
   - Click the file input or drag and drop an image file
   - Supported formats: JPG, PNG, GIF, WebP, and other common image formats

2. **View Results:**
   - The tool will automatically analyze the image
   - The top 5 most frequent colors will be displayed as color swatches
   - Each color shows its hex code below the swatch

3. **Copy Colors:**
   - Click on any color swatch to copy its hex code to your clipboard
   - Use the hex codes in your design tools or brand guidelines

4. **Image Preview:**
   - The uploaded image will be displayed for reference
   - You can see which colors were extracted from the image

## Expected Outcome

- A visual palette of the 5 most dominant colors from your image
- Hex color codes for each extracted color
- Easy copy-to-clipboard functionality for each color
- Image preview to see the source of the extracted colors

## Features

- **Automatic Extraction:** Analyzes the entire image to find dominant colors
- **Top 5 Colors:** Shows the most frequently occurring colors in the image
- **Hex Codes:** Displays the exact hex color code for each color
- **Click to Copy:** Easily copy any color code to your clipboard
- **Image Preview:** See the source image alongside the extracted palette
- **Visual Swatches:** Large color swatches for easy color identification

## Tips

- Use this tool to extract brand colors from logos or brand imagery
- Identify color schemes from inspiration images
- Create matching color palettes for design projects
- Analyze the color composition of photos or graphics
- Extract colors from competitor designs for reference
- The tool analyzes pixel frequency, so larger areas of color will appear in the palette
