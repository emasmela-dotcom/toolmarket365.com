# Content Calendar

## What It Does
A visual calendar for planning, scheduling, and managing your content posts. Supports month/week views, drag-and-drop rescheduling, and export to .ics format.

## How to Use

1. **Select view mode** - Month or Week view
2. **Click any day** to create a new post
3. **Fill in post details:**
   - Title (required)
   - Caption/notes
   - Date and time
   - Status (Draft, Scheduled, Published)
   - Platforms (tags like twitter, ig, linkedin)
4. **Drag and drop** posts to reschedule them
5. **Click existing posts** to edit or delete
6. **Search** posts using the search bar
7. **Export** to .ics file for calendar apps

## Expected Outcome

- **Visual calendar** showing all your scheduled content
- **Color-coded posts:**
  - Yellow = Draft
  - Blue = Scheduled
  - Green = Published
- **Today's date** highlighted with blue border
- **Drag-and-drop** to easily reschedule posts
- **.ics export** compatible with Google Calendar, Outlook, Apple Calendar

## Features

- **Local storage** - All data saved in your browser
- **Search** - Find posts by title, caption, or tags
- **Status tracking** - Organize by Draft/Scheduled/Published
- **Platform tags** - Track which platforms each post targets
- **Export** - Download calendar as .ics file

## Tips

- Use platform tags to filter content by social network
- Export regularly to backup your calendar
- Use status colors to quickly see what's ready to post
