# Content Idea Generator

## What It Does
Generates content ideas based on your niche. Creates titles, keywords, angles, formats, and difficulty levels for blog posts, social media, and other content.

## How to Use

1. **Enter your niche** (e.g., "vegan baking", "fitness", "travel")
2. **Select quantity** - 30 or 60 ideas
3. **Click "Generate"**
4. **Review ideas** - Each includes:
   - Title
   - Primary keyword
   - Angle (Pain-point, Myth-bust, Beginner, etc.)
   - Format (How-To, Listicle, Comparison, etc.)
   - Difficulty (Easy, Medium, Hard)
   - Alternative keywords
5. **Export options:**
   - Copy Markdown format
   - Copy CSV format
   - Download CSV file

## Expected Outcome

- **30-60 unique content ideas** tailored to your niche
- **Structured format** with title, keywords, and metadata
- **Multiple angles** - Pain-point, Myth-bust, Beginner, Expert, Case-study, Trend
- **Various formats** - How-To, Listicle, Comparison, Story, News-jack, Carousel
- **Exportable** in Markdown or CSV format

## Idea Components

- **Title**: Engaging, platform-optimized headline
- **Primary KW**: Main keyword for SEO
- **Angle**: Content approach/perspective
- **Format**: Content structure type
- **Difficulty**: Content creation complexity
- **Alt KWs**: Additional keyword suggestions

## Tips

- Be specific with your niche for better results
- Use 60 ideas for more variety
- Export to CSV for spreadsheet organization
- Use Markdown format for content planning tools
- Mix different angles and formats for content diversity
