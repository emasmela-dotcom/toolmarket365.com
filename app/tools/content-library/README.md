# Content Library

## What It Does
Organize and store all your content assets in one place. Upload images, videos, PDFs, add URLs, or create text notes. Search, filter, star favorites, and export your library.

## How to Use

1. **Add Content:**
   - **Drag & drop** files (images, videos, PDFs, docs)
   - **Click drop zone** to browse and select files
   - **Add URL** button - Paste links (Google Docs, YouTube, etc.)
   - **Add Text** button - Create text notes with title, body, and tags

2. **Organize:**
   - **Star items** - Click ⭐ to mark favorites
   - **Add tags** - When creating text items, add comma-separated tags
   - **Search** - Search by title, body, or tags
   - **Filter by tag** - Select a tag from dropdown
   - **Starred filter** - Click "⭐ Starred" to show only favorites

3. **Export:**
   - **Export CSV** - Download spreadsheet with all items
   - **Export JSON** - Download raw data file

## Expected Outcome

- **Visual library** of all your content assets
- **Preview support** for images and videos
- **Organized by tags** for easy filtering
- **Starred favorites** for quick access
- **Searchable** across all content
- **Exportable** for backup or migration

## Supported Content Types

- **Images** - JPG, PNG, GIF, etc. (with preview)
- **Videos** - MP4, MOV, etc. (with player)
- **PDFs** - Document files
- **Text files** - Any text-based file
- **URLs** - Links to external content
- **Text notes** - Custom notes with tags

## Features

- **Local storage** - All data saved in browser
- **Drag & drop** - Easy file uploads
- **Tag system** - Organize by categories
- **Star favorites** - Quick access to important items
- **Search** - Find content instantly
- **Export** - Backup your library

## Tips

- Use tags consistently for better organization
- Star frequently-used items
- Export regularly to backup your library
- Add URLs for online resources you reference
- Use text notes for ideas and reminders
- Search works across titles, bodies, and tags
