# Font Pairing Tool

## Purpose
Test and preview font combinations for headings and body text. Find the perfect typography pairing for your brand, website, or design projects by seeing how different fonts work together in real-time.

## How to Use

1. **Select Heading Font:**
   - Choose a font from the "Heading Font" dropdown menu
   - Headings typically use bold, attention-grabbing fonts

2. **Select Body Font:**
   - Choose a font from the "Body Font" dropdown menu
   - Body text typically uses readable, comfortable fonts

3. **Apply and Preview:**
   - Click the "Apply" button (or fonts update automatically)
   - View the preview section to see how your font pairing looks
   - The preview shows a sample heading and paragraph text

4. **Experiment:**
   - Try different combinations until you find the perfect pairing
   - Compare how different fonts complement each other

## Expected Outcome

- Real-time preview of font combinations
- Visual representation of how headings and body text work together
- Easy comparison of different font pairings
- Helpful guidance for choosing typography for your projects

## Features

- **12 Popular Fonts:** Access to commonly used web and system fonts
- **Real-time Preview:** See your font pairing instantly
- **Sample Text:** Preview includes both heading and body text examples
- **Easy Selection:** Simple dropdown menus for quick font changes
- **Visual Feedback:** See exactly how your chosen fonts will look together

## Tips

- **Contrast is Key:** Pair a bold heading font with a readable body font
- **Readability First:** Body text should be easy to read at smaller sizes
- **Brand Consistency:** Choose fonts that match your brand personality
- **Web Safe:** Consider using web-safe fonts for better compatibility
- **Hierarchy:** Ensure clear visual hierarchy between heading and body text
- **Test Different Sizes:** Consider how fonts look at various sizes in your actual design
