# Giveaway Manager

## Purpose
Plan and manage giveaways, track entries, and randomly select winners. Organize multiple giveaways, store participant lists, and fairly pick winners for your contests and promotions.

## How to Use

1. **Enter Prize:**
   - Enter the prize or reward for the giveaway
   - Example: "$100 Amazon Gift Card", "Free Product", "1-Year Subscription"

2. **Add Entries:**
   - List all participants/entries in the Rules/Entries field
   - Enter one entry per line (usernames, names, email addresses, etc.)
   - The tool counts entries as you type

3. **Save Giveaway:**
   - Click "Save Giveaway" to store it
   - You can manage multiple giveaways

4. **Pick Winner:**
   - Click "Pick Random Winner" on any giveaway
   - A random entry will be selected
   - Winner is saved and displayed

5. **Manage Giveaways:**
   - View all your saved giveaways
   - Delete giveaways you no longer need
   - Each giveaway tracks its own entries and winner

## Expected Outcome

- Organized list of all your giveaways
- Entry count for each giveaway
- Random winner selection
- Winner saved and displayed
- Persistent storage (saved in browser)

## Features

- **Multiple Giveaways:** Manage several giveaways at once
- **Entry Tracking:** See how many entries each giveaway has
- **Random Selection:** Fair, random winner picking
- **Winner Storage:** Winners are saved with each giveaway
- **Local Storage:** All data persists in your browser
- **Easy Management:** Delete giveaways when done

## Tips

- **Clear Entries:** Make sure entries are one per line for accurate counting
- **Save Early:** Save giveaways as you create them
- **Verify Entries:** Double-check entry list before picking winner
- **Document Winners:** Winners are saved, but you may want to screenshot
- **Multiple Rounds:** Create separate giveaways for different rounds
- **Fair Selection:** Random selection ensures fairness
- **Clean Up:** Delete completed giveaways to keep list organized

## Use Cases

- **Social Media Giveaways:** Manage Instagram, TikTok, or Twitter giveaways
- **Email Campaigns:** Track entries from email subscribers
- **Community Contests:** Organize contests in your community
- **Product Launches:** Run giveaways for product launches
- **Brand Partnerships:** Manage giveaways with brand partners
- **Engagement Campaigns:** Use giveaways to boost engagement

## Entry Formats

You can use various entry formats:
- Usernames: `@username1`, `@username2`
- Names: `John Doe`, `Jane Smith`
- Email addresses: `user@example.com`
- Any identifier that works for your giveaway

## Best Practices

- **Clear Rules:** Define giveaway rules clearly
- **Entry Deadline:** Set clear deadlines for entries
- **Verification:** Verify entries meet requirements before picking winner
- **Announcement:** Announce winners publicly for transparency
- **Follow Through:** Deliver prizes promptly after selection
