# Hashtag Analyzer

## What It Does
Analyzes the performance of hashtags you've used in your posts. Shows engagement metrics, trends, performance ratings, and provides actionable recommendations for optimizing your hashtag strategy.

## How to Use

1. **Enter your hashtags:**
   - Paste hashtags (comma or space separated)
   - Can include # symbol or not
   - Shows count of hashtags as you type

2. **Optional metrics:**
   - Number of posts (for context)
   - Total engagement (for calculations)

3. **Click "Analyze Hashtags"**

4. **Review results:**
   - Overall performance score (0-100)
   - Key metrics summary
   - Performance charts (over time, distribution)
   - Top performing hashtags
   - Complete analysis table
   - Actionable recommendations

## Expected Outcome

- **Performance Score** - Overall rating (0-100) with Excellent/Good/Needs Work
- **Key Metrics:**
  - Total hashtags analyzed
  - Average engagement per hashtag
  - Number of high performers
- **Visual Charts:**
  - Performance over time (line chart)
  - Performance distribution (pie chart)
- **Top Performers** - Top 3 hashtags with highest engagement
- **Complete Table** - All hashtags with:
  - Uses count
  - Average engagement
  - Reach
  - Performance rating (High/Medium/Low)
  - Trend (up/down with percentage)
  - Action recommendation (Keep/Test/Replace)
- **Recommendations** - Personalized advice based on your data

## Performance Ratings

- **High** - Excellent performance, keep using
- **Medium** - Good performance, test variations
- **Low** - Poor performance, consider replacing

## Action Recommendations

- **Keep** - High-performing hashtags to continue using
- **Test** - Medium performers to experiment with
- **Replace** - Low performers to swap out

## Features

- **Performance tracking** - See which hashtags work best
- **Trend analysis** - Identify growing or declining hashtags
- **Visual charts** - Easy-to-understand data visualization
- **Actionable insights** - Clear recommendations for each hashtag
- **Overall scoring** - Quick performance assessment
- **Top performers** - Highlight your best hashtags

## Tips

- Analyze hashtags regularly to track performance
- Replace low-performing hashtags with new ones
- Keep high-performing hashtags in your rotation
- Test variations of medium performers
- Use 5-10 hashtags per post for optimal reach
- Avoid using too many hashtags (10-15 max)
- Track trends to catch declining hashtags early
