# Image Alt Text Generator

## What It Does
Generates SEO-friendly alt text for images using AI. Analyzes uploaded images and creates descriptive, accessible alt text optimized for search engines and screen readers.

## How to Use

1. **Upload an image:**
   - Drag & drop image onto upload area
   - Or click to browse and select image
   - Supports all common image formats (JPG, PNG, GIF, etc.)

2. **Click "Generate Alt Text"**

3. **Review generated alt text:**
   - AI-generated description
   - Character count (optimal: under 125)
   - Copy button for easy use

4. **Copy and use** the alt text in your HTML or CMS

## Expected Outcome

- **AI-generated alt text** - Descriptive text based on image content
- **SEO-optimized** - Under 125 characters for best practices
- **Accessible** - Describes image for screen readers
- **Ready to use** - Copy directly into your code
- **Character count** - See if it meets length recommendations

## Alt Text Guidelines

- **Optimal length:** Under 125 characters
- **Be descriptive** - Explain what's in the image
- **Include keywords** - Naturally incorporate relevant terms
- **Be specific** - Avoid generic descriptions like "image"
- **Context matters** - Consider the page content
- **Skip decorative images** - Use empty alt="" for decorative elements

## Features

- **AI-powered** - Uses image-to-text AI model
- **Drag & drop** - Easy image upload
- **Character counter** - See length in real-time
- **One-click copy** - Easy to use in your workflow
- **Fallback support** - Works even if AI unavailable
- **Best practices tips** - Learn as you use

## Tips

- Upload clear, well-lit images for better results
- Review and edit generated text if needed
- Keep alt text under 125 characters
- Include relevant keywords naturally
- Describe what's important in the image
- Use for informational images, skip decorative ones
- Test with screen readers to ensure clarity
