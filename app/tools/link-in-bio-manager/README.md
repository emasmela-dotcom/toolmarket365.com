# Link in Bio Manager

## What It Does
Creates and manages a single landing page that holds all your important links. Perfect for Instagram, TikTok, and Twitter bios where you can only have one link. Organize multiple links, track clicks, and export as a static page.

## How to Use

1. **Add links:**
   - Click "Add link" button
   - Enter link title (e.g., "My Website", "Latest Blog Post")
   - Enter URL (e.g., "https://yoursite.com")
   - Click "Save"

2. **Manage links:**
   - **Click a link** - Opens in new tab and tracks clicks
   - **Edit** - Click edit icon to modify title/URL
   - **Delete** - Remove links you no longer need
   - **Drag & drop** - Reorder links by dragging

3. **Track performance:**
   - View views count (page loads)
   - View clicks count (link clicks)
   - See which links perform best

4. **Export:**
   - Click "Export static page" to download HTML file
   - Deploy to your hosting or use as-is

## Expected Outcome

- **Single bio link** - One URL to use in all your social bios
- **Multiple links** - Organize all your important links
- **Click tracking** - See which links get the most clicks
- **View tracking** - Track how many times page is viewed
- **Drag & drop ordering** - Prioritize important links
- **Exportable** - Download as static HTML page

## Use Cases

- **Instagram bio** - Replace single link with multiple options
- **TikTok bio** - One link that leads to all your content
- **Twitter/X bio** - Organize all your links in one place
- **Product links** - Multiple products, one bio link
- **Blog posts** - Link to latest posts without changing bio
- **Social profiles** - Link to all your social accounts

## Features

- **Local storage** - All links saved in browser
- **Click analytics** - Track which links perform best
- **View tracking** - See page view counts
- **Drag & drop** - Easy link reordering
- **Edit/Delete** - Full link management
- **Export** - Download as standalone HTML page
- **Profile customization** - Name, bio, avatar

## Tips

- Put most important links at the top
- Use clear, descriptive titles
- Update links regularly for fresh content
- Track clicks to see what's working
- Export and host on your domain for branding
- Use for time-sensitive links (new posts, products)
- Keep link count manageable (5-10 links ideal)
