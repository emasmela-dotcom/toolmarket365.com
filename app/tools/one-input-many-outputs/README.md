# One Input → Many Outputs

Standalone tool: one piece of content in, platform-ready copy for every selected channel out.

## What it does

- **One input:** Paste or drop text (blog, article, transcript, idea).
- **Choose platforms:** Instagram, TikTok, YouTube, Twitter/X, LinkedIn, Facebook.
- **One click:** Generate one output per platform, tuned to tone and limits.
- **Export:** Copy per platform, copy all, or download as markdown.

## How it’s different

- **Content Repurposer** — Same idea (one source → many formats); different UX and output structure.
- **Multi-Platform Generator** — Lets you pick multiple formats per platform; this tool is one output per platform, single flow.
- **This tool** — Single flow only: one input → choose platforms → one Generate → get all outputs. No format picker; one optimized output per platform.

All three tools remain separate; this one is the dedicated “one input → many outputs” experience.
