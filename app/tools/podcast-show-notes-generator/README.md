# Podcast Show Notes Generator

## What It Does
Generates professional show notes for podcast episodes from transcripts. Creates summaries, chapter timestamps, key takeaways, and SEO keywords automatically.

## How to Use

1. **Enter episode details (optional):**
   - Episode title
   - Guest name
   - Language (English, Español, Français)

2. **Add transcript:**
   - **Drag & drop** transcript file (.txt) or audio file
   - **Click drop zone** to browse and select file
   - **Paste transcript** directly into text area
   - **Paste YouTube URL** (for transcripts)

3. **Click "Generate notes"**

4. **Review generated show notes:**
   - Summary
   - Chapters with timestamps
   - Key takeaways
   - Guest information
   - SEO keywords

5. **Copy** the formatted show notes

## Expected Outcome

- **Formatted show notes** in Markdown format
- **Summary** - Condensed episode overview (25% of transcript)
- **Chapters** - Timestamped sections (auto-detected or generated)
- **Key takeaways** - Top 3 main points
- **Guest info** - Guest name if provided
- **SEO keywords** - Top 5 relevant keywords extracted
- **Ready to publish** - Copy directly to your podcast platform

## Show Notes Structure

```
# Episode Title (with Guest Name)

## Summary
[Condensed episode overview]

## Chapters
- **00:15** – Introduction topic
- **15:30** – Main discussion point
- **30:45** – Key insight
- **45:00** – Conclusion

## Key Take-aways
- Main point 1
- Main point 2
- Main point 3

## Guest Mentioned
[Guest name or N/A]

## SEO Keywords
keyword1, keyword2, keyword3, keyword4, keyword5
```

## Features

- **Automatic summarization** - Extracts key points from transcript
- **Chapter detection** - Finds timestamps or generates equal chunks
- **Keyword extraction** - Identifies top 5 SEO keywords
- **Multi-language support** - English, Spanish, French
- **File upload** - Drag & drop transcripts or audio files
- **Markdown format** - Ready for podcast platforms
- **One-click copy** - Easy to use

## Tips

- Use full transcripts for best results
- Include timestamps in transcript for accurate chapters
- Format: "15:30 – Topic name" for chapter detection
- Review and edit generated notes before publishing
- Add episode title and guest name for better formatting
- Use for consistent show notes across all episodes
- Export to Markdown for easy publishing
