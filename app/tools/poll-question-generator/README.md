# Poll / Question Generator

## Purpose
Generate engaging poll questions and discussion starters for social media. Create questions that boost engagement on Instagram Stories, Twitter polls, LinkedIn posts, and other platforms.

## How to Use

1. **Enter Topic (Optional):**
   - Enter a topic like "coffee", "fitness", "food"
   - Or leave blank for general questions
   - The tool has pre-built question pools for different topics

2. **Generate:**
   - Click "Generate" button
   - View a list of relevant questions

3. **Use Questions:**
   - Click "Copy" on any question to copy it
   - Use in Instagram Stories polls
   - Post on Twitter as polls
   - Add to LinkedIn posts for engagement
   - Use in TikTok comments or captions

## Expected Outcome

- List of 3-5 engaging questions related to your topic
- Questions optimized for social media engagement
- Easy copy-to-clipboard functionality
- Questions ready to use immediately

## Features

- **Topic-Specific Pools:** Pre-built question sets for popular topics
- **General Questions:** Default pool for any topic
- **Copy Functionality:** One-click copy for each question
- **Engagement-Focused:** Questions designed to drive interaction
- **Multiple Topics:** Coffee, fitness, food, and general categories

## Available Topics

- **Coffee:** Questions about coffee preferences and habits
- **Fitness:** Workout and health-related questions
- **Food:** Culinary preferences and dining questions
- **General:** Universal questions that work for any audience

## Tips

- **Use in Stories:** Instagram Stories polls get high engagement
- **Mix It Up:** Rotate between different question types
- **Be Relevant:** Choose topics your audience cares about
- **Timing Matters:** Post questions when your audience is most active
- **Follow Up:** Respond to poll results to boost engagement
- **Customize:** Modify generated questions to match your brand voice
- **Track Performance:** See which question types get the most responses

## Use Cases

- **Instagram Stories:** Use poll stickers with these questions
- **Twitter Polls:** Create polls using these questions
- **LinkedIn Posts:** Add questions to increase comments
- **TikTok Captions:** Include questions in video captions
- **Community Building:** Spark discussions in your community
- **Content Ideas:** Use responses as content inspiration
