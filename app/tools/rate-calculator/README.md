# Rate Calculator

## What It Does
Calculates pricing for sponsored posts, collaborations, and content creation work. Provides total rates based on hours and hourly rate, plus platform-specific recommendations based on follower count and engagement.

## How to Use

1. **Enter basic info:**
   - **Hours** - Time spent on project (e.g., 5.5 hours)
   - **Rate per Hour** - Your hourly rate (e.g., $100.00)

2. **Optional advanced info:**
   - **Followers** - Your follower count for platform rate calculation
   - **Platform** - Select Instagram, Twitter, YouTube, or LinkedIn
   - **Engagement Rate %** - Your average engagement rate

3. **Click "Calculate"**

4. **Review results:**
   - Total rate (hours × rate/hour)
   - Rate per 1K followers
   - Estimated per post (if engagement rate provided)
   - Recommended rate for platform

## Expected Outcome

- **Total Rate** - Calculated total (hours × rate per hour)
- **Rate per 1K Followers** - CPM-style calculation
- **Estimated per Post** - Based on engagement rate
- **Recommended Rate** - Platform-specific suggestion
- **Currency formatting** - All amounts in USD

## Platform Recommendations

The calculator uses industry-standard rates:
- **Instagram** - Base $100 + $0.01 per follower
- **Twitter** - Base $50 + $0.005 per follower
- **YouTube** - Base $200 + $0.02 per subscriber
- **LinkedIn** - Base $150 + $0.015 per follower

## Features

- **Simple calculation** - Hours × Rate per hour
- **Platform insights** - See recommended rates per platform
- **Follower-based** - Calculate rates based on audience size
- **Engagement-based** - Factor in engagement rate
- **CPM calculation** - Rate per 1,000 followers
- **Currency formatting** - Professional USD display

## Tips

- Use your actual hourly rate for accurate calculations
- Include all time spent (creation, editing, revisions)
- Add follower count for platform-specific rates
- Factor in engagement rate for sponsored posts
- Compare recommended rates across platforms
- Adjust rates based on project complexity
- Consider exclusivity and usage rights
- Use recommended rate as starting point for negotiations
