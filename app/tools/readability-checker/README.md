# Readability Checker

## What It Does
Analyzes text readability using Flesch Reading Ease and Flesch-Kincaid Grade Level formulas. Provides detailed statistics about word count, sentence structure, reading time, and reading level.

## How to Use

1. **Paste your text:**
   - Copy and paste any text into the text area
   - Can be articles, blog posts, social media content, etc.

2. **Click "Check Readability"**

3. **Review results:**
   - Readability Score (0-100)
   - Grade Level
   - Text Statistics
   - Advanced Metrics

## Expected Outcome

- **Readability Score** - Flesch Reading Ease (0-100)
  - 90-100: Very Easy (5th grade)
  - 80-89: Easy (6th grade)
  - 70-79: Fairly Easy (7th grade)
  - 60-69: Standard (8th-9th grade)
  - 50-59: Fairly Difficult (10th-12th grade)
  - 30-49: Difficult (College)
  - 0-29: Very Difficult (Graduate)

- **Grade Level** - Flesch-Kincaid Grade Level
  - Shows approximate school grade level
  - Ranges from elementary to graduate

- **Text Statistics:**
  - Word count
  - Sentence count
  - Paragraph count
  - Character count
  - Reading time (minutes)

- **Advanced Metrics:**
  - Average words per sentence
  - Average syllables per word
  - Total syllables

## Readability Scores Explained

**Flesch Reading Ease:**
- Higher score = easier to read
- Based on average sentence length and syllables per word
- Industry standard for content readability

**Flesch-Kincaid Grade Level:**
- Shows approximate U.S. school grade level
- Useful for targeting specific audiences
- Lower grade = easier to read

## Features

- **Flesch Reading Ease** - Industry-standard readability score
- **Flesch-Kincaid Grade** - School grade level equivalent
- **Word count** - Total words in text
- **Sentence analysis** - Sentence count and averages
- **Reading time** - Estimated reading time (200 words/min)
- **Syllable counting** - Advanced syllable analysis
- **Real-time analysis** - Instant results

## Tips

- Aim for 60-70 readability score for general audiences
- Use shorter sentences for better readability
- Target 8th-9th grade level for broad appeal
- Check readability before publishing content
- Simplify complex words for easier reading
- Break up long paragraphs
- Use readability scores to improve content
- Test different versions to optimize readability
