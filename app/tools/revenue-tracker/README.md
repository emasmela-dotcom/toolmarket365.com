# Revenue Tracker

## What It Does
Track and manage your revenue streams. Add income entries with dates, sources, and descriptions. View totals, monthly breakdowns, and export data for accounting.

## How to Use

1. **Add revenue entry:**
   - Click "Add Revenue" button
   - Enter amount ($)
   - Select date (defaults to today)
   - Enter source (e.g., "Sponsored Post", "Affiliate", "Product Sale")
   - Add description (optional)
   - Click "Add Entry"

2. **View revenue:**
   - See total revenue at top
   - View this month's revenue
   - See total number of entries
   - Browse all entries in table

3. **Filter entries:**
   - **All** - View all revenue entries
   - **This Month** - View only current month

4. **Manage entries:**
   - Delete individual entries with trash icon
   - Clear all entries (with confirmation)

5. **Export data:**
   - Click "Export CSV" to download spreadsheet
   - Includes date, amount, source, description

## Expected Outcome

- **Total Revenue** - Sum of all entries
- **This Month** - Current month's total
- **Total Entries** - Number of revenue entries
- **Detailed table** - All entries with date, amount, source, description
- **CSV export** - Downloadable spreadsheet for accounting
- **Local storage** - All data saved in browser

## Features

- **Multiple entries** - Track unlimited revenue entries
- **Date tracking** - Record when revenue was received
- **Source categorization** - Organize by income type
- **Description field** - Add notes for each entry
- **Monthly filtering** - View current month's revenue
- **CSV export** - Download for accounting/taxes
- **Local storage** - Persistent data storage
- **Delete entries** - Remove individual or all entries
