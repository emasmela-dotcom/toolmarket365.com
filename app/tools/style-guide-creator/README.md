# Style Guide Creator

## Purpose
Generate a professional style guide for your brand in Markdown format. Document your brand identity including logo, colors, and voice & tone guidelines in a shareable format that can be used across your team and projects.

## How to Use

1. **Enter Brand Information:**
   - **Brand name:** Enter your brand or company name
   - **Logo URL:** Provide a URL to your logo image (can be hosted anywhere)
   - **Primary colour:** Select your primary brand color using the color picker
   - **Secondary colour:** Select your secondary brand color using the color picker
   - **Voice & tone:** Describe your brand's voice and tone (e.g., "Friendly, concise, professional.")

2. **Build Your Style Guide:**
   - Click the "Build Guide" button
   - Your style guide will be generated in Markdown format
   - The preview box border will match your primary brand color

3. **Use Your Style Guide:**
   - **Copy:** Click "Copy" to copy the Markdown to your clipboard
   - **Download:** Click "Download .md" to save the style guide as a Markdown file
   - Share with your team or use in documentation

## Expected Outcome

- A complete Markdown style guide document
- Brand name as the main heading
- Logo image reference
- Color palette with hex codes
- Voice & tone guidelines
- Professional formatting ready for documentation
- Easy copy and download functionality

## Features

- **Markdown Format:** Generates clean, readable Markdown that works in any Markdown viewer
- **Color Pickers:** Easy selection of brand colors with hex code display
- **Visual Preview:** See your style guide formatted before copying/downloading
- **Copy to Clipboard:** Instantly copy the generated style guide
- **Download as File:** Save the style guide as a .md file
- **Brand Color Border:** Preview box uses your primary brand color

## Tips

- Use this to create consistent brand guidelines for your team
- Share the generated Markdown file with designers and content creators
- Update your style guide as your brand evolves
- Include the logo URL from your website or cloud storage
- Be specific in your voice & tone description for better consistency
- The Markdown format works in GitHub, Notion, and most documentation platforms
- You can expand the style guide manually after generation if needed
