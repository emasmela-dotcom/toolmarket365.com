# Thumbnail Text Generator

## What It Does
Generates formatted text for video thumbnails. Splits titles into multiple lines with customizable words per line, colors, fonts, and alignment. Perfect for YouTube, TikTok, and other video platforms.

## How to Use

1. **Enter title:**
   - Type or paste your video title
   - Example: "How to cook pasta in 10 minutes"

2. **Set words per line:**
   - Choose how many words appear on each line
   - Default: 3 words per line
   - Adjust based on title length

3. **Customize appearance:**
   - **Font Size** - Adjust text size (24-120px)
   - **Text Color** - Choose text color
   - **Background Color** - Choose background color
   - **Text Alignment** - Left, center, or right

4. **Click "Generate"**

5. **Review preview:**
   - See how text looks on colored background
   - Adjust settings and regenerate

6. **Copy results:**
   - **Copy Text** - Plain text format
   - **Copy HTML** - HTML div format

- **Line splitting** - Automatic word-per-line formatting
- **Visual preview** - See exact appearance
- **Color customization** - Text and background colors
- **Font size control** - Adjustable text size
- **Alignment options** - Left, center, right
- **Copy formats** - Plain text or HTML
- **Real-time preview** - See changes instantly

## Tips

- Use 2-4 words per line for best readability
- Choose high-contrast colors (light text on dark background)
- Test different alignments for your design
- Keep titles short for thumbnail readability
- Use larger font sizes (48px+) for thumbnails
- Preview before finalizing design
- Copy HTML for easy integration
- Test on actual thumbnail dimensions
