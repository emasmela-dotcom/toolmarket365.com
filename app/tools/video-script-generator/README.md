# Video Script Generator

## What It Does
Generates structured video scripts for YouTube, TikTok, Instagram Reels, and YouTube Shorts. Creates scripts with hooks, introductions, main content points, and calls-to-action based on your topic.

## How to Use

1. **Enter video topic:**
   - Type your video topic or title
   - Example: "How to cook pasta"

2. **Select platform:**
   - Choose YouTube, TikTok, Instagram Reels, or YouTube Shorts
   - Script structure adapts to platform

3. **Choose video length:**
   - **Short** - 30-60 seconds (3 sections)
   - **Medium** - 2-5 minutes (5 sections)
   - **Long** - 5+ minutes (8 sections)

4. **Click "Generate Script"**

5. **Review generated script:**
   - Hook section
   - Introduction
   - Main content points
   - Call to action
   - Outro

6. **Copy or download:**
   - Copy formatted script
   - Copy plain text
   - Download as .txt file

## Expected Outcome

- **Structured script** with sections:
  - Hook (attention-grabbing opening)
  - Introduction (welcome and topic overview)
  - Main Points (content breakdown)
  - Call to Action (engagement prompts)
  - Outro (closing)

- **Timing suggestions** - Duration for each section
- **Platform-optimized** - Tailored for selected platform
- **Ready to use** - Copy and customize as needed

## Script Structure

**Hook:**
- Attention-grabbing opening
- Teases what viewer will learn
- Duration: 0-10 seconds

**Introduction:**
- Welcome message
- Topic overview
- Sets expectations
- Duration: 30-120 seconds

**Main Points:**
- Key content sections
- Number varies by length
- Duration: 30-120 seconds each

**Call to Action:**
- Like/subscribe prompts
- Comment engagement
- Duration: 10-15 seconds

**Outro:**
- Closing message
- Next video tease
- Duration: 5-10 seconds

## Features

- **Structured sections** - Organized script flow

## Tips

- Use specific topics for better scripts
- Customize generated content to match your voice
- Adjust timing based on your speaking pace

- Keep CTAs natural and authentic
